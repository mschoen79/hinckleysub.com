## About

This is the layout design document for an N scale model railroad based on the **BNSF Hinckley Subdivision** (ex-Great Northern) and **DM&IR ore operations** in Minnesota's Arrowhead Region. The layout covers ~155 prototype miles compressed into 12 modular sections with continuous running.
